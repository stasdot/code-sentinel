# CODE SENTINEL

AI-powered code vulnerability scanner using local and cloud-based AI models.

## What it does

Scans your code for security vulnerabilities and provides recommendations on how to fix them.

## Installation

```bash
# Clone and setup
git clone https://github.com/yourusername/code-sentinel.git
cd code-sentinel

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
# Scan a project
python -m src.scanner scan ./my-project
```
